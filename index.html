YourForm

This is a term project for Software Project Management course. YourForm is a data collection engine. It builds engaging interfaces for users to answer questions that may concern their wellbeing and provides probabilistic answers to such questions, while also collecting important data regarding the setting. During the course, we implemented Cervical Cancer and Heart Stroke prediction ML models. After completing surveys, users could get a prediction of the probability of those diseases.
Front-end: React
Back-end: Node.js + MongoDB
GCP ML Engine was used to deploy the models.
